update notice: The latest version supports Anki's built-in night mode which was redesigned with Anki version 2.1.20. So this add-on also requires at least 2.1.20 now. This add-on loads the content of a field from the editor into a new dialog with the wysiwyg html editor tinymce. To open this editor click the "T5" button or use the shortcut (adjustable from the add-on config). This editor offers some advanced editing options especially with regard to tables: You can add or delete rows or columns. You can style the table. You can merge cells. You can style the borders and background colors, etc. Drag and Drop of content is not supported. You can paste into this dialog. But I would only paste text from my pdf reader or text editor. I wouldn't paste content from websites because (contrary to the Anki editor) images are not downloaded and the html is not cleaned. the html is not cleaned this might cause problems and/or vulnerabilities in the long run. General point: Use this at your own risk. If you find a problem, bug, or have a bug fix or improvement let me know in the official anki support forum, on github or on reddit (/u/ijgnord). Thanks @/u/glutaniamte for pointing me to a way to display the images. Configuration see the readme in the add-on configuration dialog.
