This add-on loads the content of a field from the editor into a new dialog with the wysiwyg html editor tinymce. To open this editor click the "T5" button or use the shortcut (adjustable from the add-on config). In the config you can also select another editor like tinymce4, ckeditor4. This editor offers some advanced editing options especially with regard to tables: You can add or delete rows or columns. You can style the table. You can merge cells. You can style the borders and background colors, etc. Drag and Drop of content is not supported. You can paste into this dialog. But I would only paste text from my pdf reader or text editor. I wouldn't paste content from websites because (contrary to the Anki editor) images are not downloaded and the html is not cleaned. the html is not cleaned this might cause problems and/or vulnerabilities in the long run. You can apply custom css to the editor in the extra dialog by putting it into a file named "_tinymce_custom.css" in the media folder of your profile. I don't need this so I don't know how this interacts with the tinymce skin and theme I use. General point: Use this at your own risk. If you find a problem, bug, or have a bug fix or improvement let me know in the official anki support forum, on github. Thanks @/u/glutanimate for pointing me to a way to display the images. # Configuration see the readme in the add-on configuration dialog. You can also adjust the config files of the editors in the add-on folder. Keep in mind that updates of this add-on overwrite all changes in the add-on folder. So if you adjust files in the add-on folder keep backups and remember to reapply them after updating this add-on. # Old Anki Versions If you have an older Anki version and can't update to a newer one you could try out an old version of this add-on which you can download as an ankiaddon from here.
